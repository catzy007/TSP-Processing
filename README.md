# TSP-Processing
Brute-force TSP (Lexicographic Ordering) and Pathfinder (A-STAR) algorithm to find (best) route to minimise distance in real-world local map.

* Draw your point 
    * `min : 4`
    * `max : 15`
* Press red button to begin
